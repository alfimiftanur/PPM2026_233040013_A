## 🚀 Cara Menjalankan

Pastikan Flutter SDK sudah terinstall di komputer kamu. Cek dengan:

```bash
flutter --version
```

### Menjalankan project:

```bash
# Masuk ke folder project yang ingin dijalankan
cd latihan_widget
# atau
cd PPM2026_233040013_C

# Install dependencies
flutter pub get

# Jalankan aplikasi
flutter run
```
